## Web Server

* [Redbird](https://github.com/OptimalBits/redbird)

  * BSD 2-Clause "Simplified" License.
* [Caddy](https://github.com/mholt/caddy) - [Site](https://caddyserver.com/)
  * Fast, cross-platform HTTP/2 web server with automatic HTTPS.
  * Apache License 2.0
